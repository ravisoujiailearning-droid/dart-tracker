Host DART on GitHub Pages

Free hosting in 5 minutes. Both phones get a permanent URL that always works.

1

Create a free GitHub account

Go to github.com/signup and create an account (if you don't have one already). It's free.

2

Create a new repository

After logging in, click the + icon (top right) → New repository.

  • Repository name: dart-tracker
  • Set it to Public
  • Check "Add a README file"
  • Click Create repository
3

Upload your files

Inside your new repository, click Add file → Upload files.

Upload these 3 files (from your DART download):

  • index.html
  • setup-guide.html
  • google-apps-script.js

Scroll down and click Commit changes.

4

Enable GitHub Pages

In your repository, click Settings (top menu) → scroll down to Pages (left sidebar).

  • Under "Source", select Deploy from a branch
  • Branch: main — folder: / (root)
  • Click Save

Wait about 1 minute. Your app is now live at:

https://YOUR-USERNAME.github.io/dart-tracker/
💡 GitHub will show you the exact URL in the Pages settings section once it's ready.
5

Open and configure

Open the URL on your phone. You'll see the setup form. Fill it in once:

  • Your name, cousin's name
  • Google Sheet ID
  • Apps Script URL

Now this is saved permanently in your phone's browser. Every time you visit the URL, it goes straight to the tracker — no form again.

💡 Add to home screen: Chrome → 3-dot menu → "Add to Home Screen". Opens like an app!
6

Share with your cousin

Just send them the same URL on WhatsApp:

https://YOUR-USERNAME.github.io/dart-tracker/

They open it, fill in the setup form once (same Sheet ID and Script URL), and they're in. Both of you now write to the same Google Sheet. ✅

⚠️ Make sure your cousin enters the same Google Sheet ID and Script URL as you — that's what connects you both to the same data.
7

View your raw data in Google Sheets

Open your Google Sheet anytime. You'll see a sheet tab called DARTData. Every entry is stored there as a row — one row per day, with all activity states saved as JSON.

You can always see exactly what's been saved, export it, or back it up.
